# node-restapi-template

express, passport, mongodb on server

ionic2 with sidemenu template on client

# usage

* `git clone https://github.com/waxa/node-restapi-template.git project`
* `cd project`
* `npm install`
* `cd app/client`
* `npm install`
* `npm run ionic:build`
* `cd ../../`
* `npm start`
